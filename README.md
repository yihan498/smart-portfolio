## 产品简介

**智配**是一套个人投资组合量化配置模型，通过11个因子综合评估你的风险承受能力，结合均值-方差优化（Markowitz MVO），给出四类资产的最优配置比例，摆脱"凭感觉投资"或"照搬别人方案"的困境。

## 四类资产

| 资产 | 代表 | Sharpe 比率 | 特点 |
|------|------|------------|------|
| 超级进攻型 | 比特币（BTC） | 0.213 | 极高波动，低相关性，4年减半周期 |
| 稳健进攻型 | 美股优质个股 | **0.333** | Sharpe 最高，长期复利核心 |
| 中型配置 | 中国高分红/国债 | 0.231 | 低相关性分散化工具 |
| 防守型 | 美元存款/货基 | — | 保本保息，流动性储备 |

> 美股始终是最大仓位核心，BTC 的价值在于与其他资产的低相关性（ρ = 0.15–0.30），通过分散化提升整体组合效率。

## 三步计算流程

```
Step 1  →  11个因子打分  →  加权求和得 RS 原始值（0–10）
Step 2  →  三层约束修正  →  RS_final = min(RS + 青年加速, 年龄上限, 财富保全上限)
Step 3  →  目标波动率映射  →  σ = 1% + 34% × (RS/10)^1.5  →  MVO 查表得配置比例
```

## 11个评分因子

| 因子 | 权重 | 说明 |
|------|------|------|
| F1 年龄 | 20% | 越年轻风险承受能力越强 |
| F2 收入储蓄比 | 15% | 储蓄率越高越有底气冒险 |
| F3 资产充裕度 | 10% | 资产能覆盖多少年支出 |
| F4 紧急储备金 | 10% | 应急资金是否充足 |
| F5 负债压力 | 10% | 负债越少越灵活 |
| F6 家庭负担 | 10% | 单身 vs 多孩家庭 |
| F7 收入稳定性 | 5% | 就业类型稳定性 |
| F8 市场周期 | 5% | 熊市底部=高分，牛市顶部=低分 |
| F9 经济周期 | 5% | 宏观经济所处阶段 |
| F10 投资经验 | 5% | 实际操盘经验年限 |
| F11 主观风险意愿 | 5% | 个人主观风险偏好 |

## 三层约束系统（v2.2）

本模型在11因子基础上叠加三层独立约束，确保配置结果符合生命周期规律：

- **层②a 青年加速**：35岁以下额外奖励最多 +1.07，鼓励年轻时积累风险资产
- **层②b 年龄上限**：`age_cap = max(0.5, 10 × ((80-age)/55)^0.7)`，80岁时强制降至 RS ≤ 0.5
- **层③ 财富保全**：资产覆盖年数超过50年后逐步压低 RS 上限，巨额资产优先保值
