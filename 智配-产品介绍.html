工作原理

三步，从个人情况到最优配置

每一步都有数学依据，配置比例是算出来的，不是定出来的

1
评估个人情况
→ RS得分（含年龄联动）
对11个因子加权求和得到RS原始分，再经过年龄联动调整：年龄越大上限越低，35岁以下有加速奖励。
RS原始 = 0.20×F1 + 0.15×F2 + ...

年龄上限 = max(0.5, 10×((80-年龄)/55)^0.7)
青年加速 = (35-年龄)/35×1.5 [35岁以下]

RS最终 = min(RS原始+青年加速, 年龄上限)
2
RS得分
→ 目标波动率
将RS得分换算为你能承受的年化组合波动率σ，作为优化的风险约束。
σ = 1% + 34% × (RS/10)^1.5

RS=0 → σ=1%（近现金）
RS=5 → σ=13%（60/40均衡）
RS=10 → σ=35%（激进）
3
均值-方差优化
→ 最优配置
在σ_p ≤ σ_target约束下，用二次规划求使期望收益最大的权重组合。
max w·μ （最大化收益）
s.t. √(w'Σw) ≤ σ_target
Σwᵢ = 1
w_BTC ≤ 40%
架构原理

双引擎架构：主观判断 × 数学优化

两个系统各司其职，缺一不可

引擎一
11因子打分系统
将你的个人情况——年龄、收入、资产、家庭结构、市场环境——转化为一个可量化的风险预算。这是模型的"人性化"接口，让主观的生活状态变成客观的数字RS得分。
RS = Σ(wᵢ × Fᵢ), i = 1..11
权重之和 = 100%
每项因子分值 ∈ [0, 10]
→ 输出：RS得分（0-10）
引擎二
均值-方差优化（MVO）
给定你的风险预算σ_target，在四类资产中寻找使期望收益最大的权重组合。现代投资组合理论的核心框架，确保每一份风险都被最高效地利用。
max w · μ
s.t. √(w'Σw) ≤ σ_target
Σwᵢ = 1, wᵢ ≥ 0
→ 输出：最优配置比例
RS得分  →  σ_target = 1% + 34% × (RS/10)^1.5  →  MVO优化  →  最优配置
为什么这个组合有效？ 打分系统解决了"你能承受多少风险"的问题——这需要人的判断，因为生活状态无法被算法直接读取。MVO解决了"给定这个风险上限，如何最大化收益"的问题——这不需要主观判断，有严格的数学最优解。两者分工明确，前者依赖你的诚实自评，后者依赖诺贝尔奖级别的数学框架，组合起来才是完整的决策工具。

模型局限性

理性使用，了解边界

⚠
参数不确定性
μ（预期收益）、σ（波动率）均为历史数据和主观估计，未来实际表现可能大幅偏离。BTC历史仅约10年，参数估计误差尤其大。
⚠
正态分布假设
MVO假设收益率正态分布，但BTC等资产存在肥尾风险（极端涨跌）。实际极端情境下，组合表现可能比模型预测更差。
⚠
相关性时变
各资产相关性会随市场环境变化，尤其在危机时期往往趋向于1（同步下跌），此时分散化效果大幅减弱。
✓
建议使用方式
每年评估一次，市场周期变化时季度微调。将模型结果作为决策起点，结合个人判断使用。本工具不构成投资建议。
